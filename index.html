Relative units

Relative units are one of the main powerful features of CSS : thank to them, one can easily display responsive and accessible layout. Let's review the most used and interesting relative units.

EM

When should we use em ?

As you can see with the example above, the em unit can be tricky. It best to use em for typography and spacing on textual components, like a button or input fields.
